# TRELLO – Részletes, Átfogó Dokumentáció

A Trello egy könnyen használható, intuitív, kártya-alapú projektmenedzsment eszköz. Ideális kisebb csapatoknak vagy egyszerűbb folyamatokhoz.

## Trello alapelvek

A Trello a vizuális Kanban koncepcióból épül:

- board → lista → kártya
- drag & drop
- könnyű használat
- minimális konfiguráció
- real-time szinkron

## Struktúra

Alap felépítés:

Board
└── List
└── Card
└── Checklist

### Board

Egy teljes projekt.

### List

Folyamat lépések (pl. "To Do / Doing / Done").

### Card

A feladatot jelöli.

### Checklist

Kisebb alfeladatok.

## Funkciók

### Kártyák tartalma

- leírás
- tag-ek
- határidő
- mellékletek
- címkék (labels)
- kommentek
- aktivitás log

### Power-Ups

Funkcióbővítők, pl:

- Calendar
- Timeline
- Slack kapcsolat
- Google Drive
- GitHub integráció

## Automatizálás (Butler)

A Trello egyszerű, logikus automatizmust ad:

- amikor → ha → akkor
- trigger → condition → action

Példák:

- új kártya mozgása listák között
- automatikus címkézés
- értesítések
- időzítések

## Nézetek

A Trello egymás után vezette be az új nézeteket:

- Board view
- Calendar view
- Table view
- Timeline
- Dashboard

Ez a verzió felnőtt a Jira-hoz funkcionalitásban – mégis egyszerű.

## Előnyök

- intuitív
- gyors betanulás
- vizuális rendszer
- mobilbarát
- real-time synchronizáció
- könnyű megosztás

## Hátrányok

- korlátozott testreszabhatóság
- nem ideális nagy csapatoknak
- permission rendszer egyszerű
- komplex projektekhez kevés
